# BMI Calculator (C++)

A beginner-level C++ project that calculates Body Mass Index (BMI) using
Object-Oriented Programming concepts.

## Features
- Takes user height and weight as input
- Calculates BMI
- Displays health category (underweight, normal, overweight)
- Suggests whether to gain or lose weight

## OOP Concepts Used
- Encapsulation (data hiding using classes)
- Inheritance (BMI_Calculator inherits from Weight, Height, BMI_category and normalBMI)

## Formula Used
BMI = weight (kg) / height² (m²)
